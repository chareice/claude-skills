# chareice-skills

Claude Code skills by chareice for DevOps, testing, and productivity.

## Installation

```bash
/plugin marketplace add chareice/claude-skills
/plugin enable chareice-skills
/reload-plugins
```

## Skills

| Skill | Description |
|-------|-------------|
| devops | Server deployment, diagnostics, logs, rollback, and environment setup |
| e2e | E2E testing: run, fix, write, list, suggest, setup, teardown |
| openapi-generator | Generate and sync OpenAPI/AsyncAPI documentation |

## Slash Commands

| Command | Description |
|---------|-------------|
| /ch-memory-analysis | Investigate memory usage in ClickHouse |
| /ch-slow-query-analysis | Analyze slow queries in ClickHouse |
| /cleanup-branches | Clean up local branches and worktrees |
